Detailed Description

Function provides C bindings for thorvg library. Please refer to src/examples/Capi.cpp to find thorvg_capi examples.

thorvg_capi module provides a set of functionality that allows to implement thorvg C client with following funcionalities

  • Drawing primitive paints: Line, Arc, Curve, Path, Shapes, Polygons
  • Filling: Solid, Linear, Radial Gradient
  • Scene Graph & Affine Transformation (translation, rotation, scale ...)
  • Stroking: Width, Join, Cap, Dash
  • Composition: Blending, Masking, Path Clipping, etc
  • Pictures: SVG, Bitmap
