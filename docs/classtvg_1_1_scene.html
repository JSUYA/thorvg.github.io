Detailed Description

A class enabling to hold many Paint objects.

As a whole they can be transformed, their transparency can be changed, or the composition methods may be used to all of them at once.

◆ reserve()

Result tvg::Scene::reserve ( uint32_t  size)
noexcept

Sets the size of the memory block, where all the paints pushed into the scene are stored.

If the number of objects pushed into the scene is known in advance, calling the function prevents multiple memory reallocation, thus improving the performance.

Parameters
