◆ transform()

Result tvg::Paint::transform ( const Matrix &  m)
noexcept

Sets the matrix of the affine transformation for the object.

The augmented matrix of the transformation is expected to be given. The elements e11, e12, e21 and e22 represent the rotation matrix, including the scaling factor. The elements e13 and e23 determine the translation of the object along the x and y-axis, respectively. The element e31 and e32 are set to 0, e33 is set to 1.

Parameters
[in]mThe 3x3 augmented matrix.
Returns
Result::Success when succeeded, Result::FailedAllocation otherwise.
