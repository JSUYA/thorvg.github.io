Detailed Description

A abstract class for managing graphical elements.

Basically a graphical elements in TVG is composed as an object in a scene (Canvas). Paint represents a graphical object, suggest common behaviors such as duplication, transformation and composition, etc. TVG recommends user to regard a paint as a pack of volatile commands, they can prepare a paint then request Canvas to run it.
